## How to run
`export PATH=$PATH:/usr/local/go/bin`
- `hugo server` from the main directory ayushbits.github.io

- To deploy static pages in public/ - run `hugo`
** It will deploy the current code to public/

- public/ needs to be pushed to github after running `hugo' command. 
- We can add public/ folder to the current project and commit the changes directly to github

or 

- public/ can be pushed to github using terminal (since we cannot push from the parent folder)
git add .
git commit -m "message here"
git push origin HEAD:master

- wait for some time to reflect changes in ayushbits.github.io
